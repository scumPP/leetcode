# LC 96 不同的二叉搜索树

## 题目

给定一个整数 n，求以 1 ... n 为节点组成的二叉搜索树有多少种？

示例:

```c++
输入: 3
输出: 5
解释:
给定 n = 3, 一共有 5 种不同结构的二叉搜索树:

1         3     3    2    1
 \       /     /    / \    \
  3     2     1    1   3    2
 /     /       \             \
2     1         2             3
```

## 思路：
动态规划思想   

- 假设此时序列为n，以i节点为根，那么所得的二叉搜索树的个数为$F(i,n)$

- 假设最后的结果为$G(n)$，即所有不同结构的二叉搜索树个数

- 通过推导可得以下两个公式：   

   - $F(i,n)=G(i-1)*G(n-i)$

   - $G(n)=\displaystyle\sum_{i=1}^nG(i-1)*G(n-i)$ 

具体代码为：

```c++
int numTrees(int n) {
    vector<int> res(n+1,0);
    res[0]=1;
    res[1]=1;
    for(int i=2;i<n+1;i++){
        for(int j=1;j<i+1;j++){
            res[i]+=res[j-1]*res[i-j];
        }
    }
    return res[n];
}
```
